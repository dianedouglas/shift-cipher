This is My Custom Shift Cipher

Also known as a Caesar Cipher

The phrase you enter below will be put into code, and the code is created by the shift value you enter.

A shift value of 1 will move each letter and number forward by 1 space.

For example:

"A" shift + 1 = "B"

"Z" shift + 1 = "A"

"1" shift + 1 = "2"

"9" shift + 1 = "0"

"5" shift + 22 = "7"
